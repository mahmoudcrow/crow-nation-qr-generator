# Contributor Covenant Code of Conduct (short)

We value the participation of every contributor and want all interactions to be welcoming and respectful.

Expected behavior
- Be respectful and considerate in language and actions.
- Assume good intent and seek to resolve conflicts constructively.

Unacceptable behavior
- Harassment, discrimination, or abusive language will not be tolerated.

Reporting
- If you experience or witness behavior that violates this code, please open a private issue or contact the maintainers.
