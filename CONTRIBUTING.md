# Contributing

Thank you for considering contributing to Crow Nation QR Generator.

How to contribute
- Open an issue to propose features or report bugs.
- Fork the repo and create a feature branch: `git checkout -b feature/my-change`.
- Keep PRs small and focused; include screenshots or steps to reproduce when relevant.

Coding style
- Follow WordPress PHP coding standards where practical.
- Use clear commit messages and reference related issue numbers.

Pull request checklist
- [ ] Code builds and runs without errors
- [ ] No new PHP warnings or notices
- [ ] Documentation updated (if applicable)
